Управление медиабиблиотекой ​

Инструкции по решению базовых задач в панели Медиабиблиотека с примерами.

🧭 Ориентация по папкам ​

Навигация по медиабиблиотеке работает тем же образом, что и навигация по папкам внутри Проводника Windows. Организация папок была выстроена таким образом, что каждая папка отвечает за хранение медиафайлов конкретной страницы, например, папка Педагогический состав отвечает за хранение медиафайлов страницы Педагогический состав и т. д.

Организация папок может быть выстроена как угодно.

При открытии вкладки Медиабиблиотеки вас встречает набор корневых папок, которые организуют ресурсы по контенту страниц сайта.

Для того чтобы перемещаться по внутренностям папки, вам нужно на неё нажать. После нажатия окно сменится, а ваш текущий маршрут, который находится в шапке окна, покажет ваше текущее положение.

Пример навигации

Дана вложенная структура, в которой есть папка A, внутри неё B, а внутри B папка C.

В папке C, под заголовком Медиабиблиотека, можно увидеть текущее расположение в папке C. Текущая открытая папка подсвечена жирным шрифтом. Можно нажать на папку B внутри окна навигации, тогда откроется папка B, либо нажать на три точки, идущие до неё, тогда откроется список всех предыдущих папок.

Можно нажать на кнопку <- Back, тогда откроется предыдущая папка.

📁 Создание папки ​

Для того чтобы создать папку, нажмите на кнопку Добавить папку в правом верхнем углу. Перед вами откроется окно с двумя полями ввода — названием папки и местом её сохранения. По умолчанию место сохранения будет текущая открытая папка, её название будет стоять по умолчанию.

Пример создания папки с указанием пути

  1. Перейдите в Медиабиблиотеку;

  2. Перейдите в папку Наша команда;

  3. Нажмите на кнопку Добавить папку. В открывшемся меню напишите имя папки в поле Name, а в поле Location выберите имя папки, внутрь которой будет создана папка. По умолчанию путь создания будет являться текущая открытая папка. Самой верхней папкой считается Media Library, она содержит в себе все папки, которые вы видите при первом открытии Медиабиблиотеки;

  4. Нажмите на кнопку Создать и перейдите в корневую папку Медиабиблиотеки. В ней должна появиться новая папка.

📨 Загрузка файлов на сервер ​

Для того чтобы загрузить ресурс, нажмите на кнопку Добавить ресурсы в правом верхнем углу. Перед вами откроется окно с полем, куда можно перетащить нужные вам файлы, либо нажмите на кнопку Обзор файлов и выберите нужные из вашего компьютера. Можно сохранить файл как ссылку, тогда откройте C URL и вставьте ссылку на ресурс.

После добавления ресурса у вас появится окно Добавить ресурс, в нём появятся все файлы, которые были выбраны для загрузки, а также две кнопки: Добавить ресурсы — позволяет выбрать дополнительные файлы для загрузки и Загрузить ресурс — для отправки файлов на сервер.

Пример добавления фотографии сотрудника

  1. Перейдите в Медиабиблиотеку;

  2. Перейдите в папку Наша команда;

  3. Нажмите на кнопку Добавить ресурс. В открывшемся окне перетащите нужное фото в центр квадрата или нажмите на кнопку Обзор файлов и выберите нужный файл фотографии. Если нужно прикрепить файл как ссылку, то перейдите во вкладку С URL и укажите адрес фотографии;

  4. После добавления файла у вас откроется окно состояния загрузки. Вы можете добавить дополнительные фотографии в случае, если нужно добавить несколько новых сотрудников. После добавления нажмите на кнопку Загрузить 1 ресурс в библиотеку;

  5. В папке Наша команда появился новый файл с фотографией.

🗄️ Изменение содержимого медиабиблиотеки ​

Для того чтобы изменить ресурс или папку, вам нужно навести на неё курсор мыши. Внутри ресурса или папки справа появится кнопка Редактировать папку с символом карандаша. После нажатия на кнопку перед вами откроется окно, оно отличается у ресурса и папки, но в общем виде с помощью этого окна можно посмотреть информацию о папке или ресурсе, а также удалить или изменить данные.

Изменение папки

  1. Перейдите в Медиабиблиотеку;

  2. Нажмите на кнопку Редактировать папку папки Спортивные секции;

  3. В открывшемся окне измените поле Name и Location. Далее нажмите на кнопку Save;

  4. Перейдите в папку Актуальные предложения, в ней должна появиться изменённая папка.

Удаление папки

  1. Перейдите в Медиабиблиотеку;

  2. Перейдите в папку Актуальные предложения;

  3. Наведите на кнопку с тремя точками в правой части папки и нажмите на них;

  4. Нажмите на кнопку Delete folder;

  5. Нажмите на кнопку Подтвердить;

  6. В папке Актуальные предложения должна пропасть папка Спортивные мероприятия.

Изменение ресурса

  1. Перейдите в Медиабиблиотеку;

  2. Перейдите в папку Наша команда;

  3. Нажмите на кнопку Редактировать;

  4. Нажмите на кнопку Заменить ресурс и в открывшемся файловом менеджере выберите нужное фото;

  5. Нажмите на кнопку Готово;

  6. Проверьте, что ресурс был изменён.

Удаление ресурса

  1. Перейдите в Медиабиблиотеку;

  2. Перейдите в папку Наша команда;

  3. Нажмите на кнопку Редактировать;

  4. Нажмите на кнопку Удалить;

  5. Нажмите на кнопку Подтвердить;

  6. Проверьте, что ресурс был удалён.
