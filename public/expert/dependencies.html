Зависимости ​

Описание основных используемых технологий и файлов конфигурации для формирования общего представления о созданном решении.

📋 Обзор зависимостей ​

Более подробно можно ознакомиться в package.json файле в корневой папке проекта.

Окружение
  • Typescript ~5.8.3 - язык программирования
  • Node.js ~22.16.0 - серверная среда выполнения JS кода
Frontend
  • Vite ^6.3.5 - сборщик проекта
  • VitePress ^1.6.3 - документирование решения
  • Eslint ^9.25.0 - проверка на ошибки до исполнения кода
  • Prettier ^3.5.3 - форматирование кода
  • React ^18.0.0 - создание UI компонентов
  • React DOM ^18.0.0 - отрисовка react-компонентов в браузере
  • React Router ^6.0.0 - маршрутизация на стороне клиента
  • Tailwind CSS ^4.1.8 - стилизация компонентов
  • Radix UI ^1.1.14 - использование готовых компонентов
  • Axios ^1.10.0 - написание запросов на сервер
Backend
  • Strapi 5.16.0 - система управления контентом
